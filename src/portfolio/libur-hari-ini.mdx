A simple web app to check if today is a holiday in Indonesia. It uses the [Official Announcement](https://kemenkopmk.go.id/sites/default/files/pengumuman/2022-10/SKB%20LibNas%20dan%20CuBer%202023.pdf) to determine the holiday. The app is built with NextJs, TypeScript, Styled-Components, TailwindCSS, and deployed on Vercel. The data is stored in a JSON file.
